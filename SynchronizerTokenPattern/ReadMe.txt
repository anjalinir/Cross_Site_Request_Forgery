*------------------------------------------------------*
*---CSRF Protection using Synchronizer Token Pattern---*
*------------------------------------------------------*

This project is about how Cross-Site Request Forgery Protection can be done by using Synchronizer Token Pattern

Web Application Programming Language: ASP.Net C#

Steps to run
____________

* Launch the developed project using Visual Studio 2015 or above version.
* Start the program after selecting STP Web Application.
* It will prompt a login page. Use creadentials as Username: 'Test' and Password: '123' for demostration purposes.
* Once you login you will redirect to the Home.aspx page. 
* By clicking the button placed on the Home page, it will validate your request is success or not.
* The web application can access via localhost.
